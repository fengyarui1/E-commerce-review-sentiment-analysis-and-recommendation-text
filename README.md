# E-commerce-review-sentiment-analysis-and-recommendation-text
本项目是一个基于Python的电商评论情感分析与推荐文本生成系统，主要功能包括：  1. **情感分析**：对用户输入的文本进行情感极性分析，判断其为正面、负面还是中性情感 2. **好评生成**：根据用户输入的文本内容，提取关键词并生成符合要求的好评文本  系统采用了朴素贝叶斯分类器进行情感分析，并结合TextRank算法和星火大模型实现好评生成功能。

## 项目结构

```
电商评论情感分析与推荐文本生成系统/
├── data/                  # 训练数据
│   ├── pos/               # 正面评论
│   └── neg/               # 负面评论
├── models/                # 训练好的模型
├── .venv/                 # 虚拟环境
├── comment_generation.py  # 好评生成模块
├── data_cleaning.py       # 数据清洗模块
├── interface.py           # 主界面
├── model_training.py      # 模型训练模块
├── sentiment_predictor.py # 情感分析预测器
├── stopwords.txt          # 停用词表
└── 测试文本               # 测试文本文件
```

## 核心功能模块

### 1. 情感分析模块

**功能**：对输入文本进行情感极性分析，返回情感类别（正面、负面、中性）和置信度。

**实现**：
- 使用朴素贝叶斯分类器（MultinomialNB）作为基础模型
- 采用TF-IDF特征提取方法
- 支持中性情感判断（通过设定阈值）

**核心代码**：`sentiment_predictor.py`

### 2. 好评生成模块

**功能**：根据输入文本提取关键词，并生成符合要求的好评文本。

**实现**：
- 使用TextRank算法提取关键词
- 调用星火大模型生成好评文本
- 支持自定义关键词数量和词性筛选

**核心代码**：`comment_generation.py`

### 3. 数据清洗模块

**功能**：清洗文本数据，去除噪声，准备训练数据。

**实现**：
- 去除HTML标签、URL和特殊字符
- 使用jieba分词
- 去除停用词和单字符

**核心代码**：`data_cleaning.py`

### 4. 模型训练模块

**功能**：训练情感分析模型并保存。

**实现**：
- 加载和清洗数据
- 划分训练集和测试集
- 构建模型管道（TF-IDF + 朴素贝叶斯）
- 评估模型性能
- 保存训练好的模型

**核心代码**：`model_training.py`

### 5. 主界面模块

**功能**：提供用户交互界面，展示情感分析结果和生成的好评。

**实现**：
- 使用tkinter构建GUI
- 支持文本输入和结果展示
- 提供情感分析和好评生成按钮

**核心代码**：`interface.py`

## 技术栈

- **前端**：tkinter
- **后端**：Python 3.x
- **情感分析**：scikit-learn（朴素贝叶斯分类器、TF-IDF）
- **文本处理**：jieba（分词、TextRank）
- **好评生成**：星火大模型API
- **数据存储**：joblib（模型序列化）

## 系统流程

1. **情感分析流程**：
   - 用户输入文本
   - 系统清洗文本并分词
   - 使用训练好的模型预测情感
   - 展示情感分析结果

2. **好评生成流程**：
   - 用户输入文本
   - 系统提取关键词
   - 生成请求内容
   - 调用星火大模型生成好评
   - 展示生成的好评

## 模型训练

1. **数据准备**：
   - 正面评论数据：存放在`data/pos/`目录
   - 负面评论数据：存放在`data/neg/`目录
   - 停用词表：`stopwords.txt`

2. **训练步骤**：
   - 运行`model_training.py`脚本
   - 系统会自动加载数据、训练模型并保存到`models/sentiment_model.pkl`

3. **模型评估**：
   - 训练过程中会输出准确率和F1分数
   - 默认测试集比例为20%

## 使用说明

1. **启动系统**：
   ```bash
   python interface.py
   ```

2. **情感分析**：
   - 在左侧文本框输入要分析的文本
   - 点击"预测情感"按钮
   - 右侧文本框会显示情感分析结果

3. **好评生成**：
   - 在左侧文本框输入要生成好评的文本
   - 点击"生成好评"按钮
   - 右侧文本框会显示生成的好评
